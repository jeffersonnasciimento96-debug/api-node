# ⚡ API REST com Node.js

## 📖 Descrição
API RESTful construída em **Node.js** para manipulação de dados via endpoints.  
Inclui autenticação JWT e integração com banco de dados.

## 🚀 Tecnologias
- Node.js / Express
- JavaScript
- SQL (PostgreSQL/MySQL)
- Git/GitHub

## ▶️ Como Executar
1. Clone o repositório
2. Instale dependências: `npm install`
3. Execute: `npm start`
